# MGP_2526
 The assessment repo for 2526

 Controls:
 WASD to move
 Left click to grapple

 Any of the red cubes can be grappled to, or any object with the default layer GrapplePreset.
 
 The UI indicates if you can grapple, and how many you have left:
 Grey - no grapple/grapple target.
 Red - 1
 Orange - 2
 Green - 3
